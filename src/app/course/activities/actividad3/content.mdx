
# Actividad 5: Implemente una jerarquía de clases basada en la relación entre un vehículo y sus tipos (coche, camión, autobús y motocicleta). Utilice la herencia para diseñar una superclase Vehiculo y cuatro subclases Coche, Camion, Autobus, y Motocicleta.

1.  Crear una clase Vehiculo como la superclase. Debe tener atributos como marca, modelo, velocidadMaxima, y capacidadDeCombustible. Incluya métodos para mostrar los detalles del vehículo, acelerar, frenar y encender el vehículo.

2.  Crear cuatro subclases (Coche, Camion, Autobus, y Motocicleta) que hereden de la clase Vehiculo. Sobrescriba y amplíe las funcionalidades de la superclase según corresponda para cada subclase.
    -   Para la clase Coche, incluya un atributo color y un método para abrir y cerrar las puertas del coche.
    -   Para la clase Camion, incluya un atributo capacidadDeCarga y un método para cargar y descargar mercancías.
    -   Para la clase Autobus, incluya un atributo capacidadMaximaDePasajeros y un método para abordar y bajar pasajeros.
    -   Para la clase Motocicleta, incluya un atributo numeroDeCilindros y un método para activar y desactivar las luces delanteras y traseras.

3.  Cada subclase debe implementar el método mostrarDetalles() para mostrar la información específica de cada tipo de vehículo.

4.  En la clase Main, cree un objeto de cada subclase y demuestre las funcionalidades que implementó en cada una de ellas.
