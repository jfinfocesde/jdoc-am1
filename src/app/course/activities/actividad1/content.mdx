## Actividad: Apliación en Java de lógica de programación

### Instrucciones:

Desarrollar un programa en Java que:

1.  Solicite información personal de una persona (nombre, edad, ciudad).
2.  Solicite una contraseña y la valide (al menos 8 caracteres, una letra mayúscula y un número).
3. Validación de la contraseña: 
    - Si la contraseña es válida:
            - Imprimir mensaje de bienvenidad utilizando la información personal de la persona
            - mostrar un menú de comidas que se encuentra en un arreglo utilizando bucles (for, while o do while).
            - Permitir al usuario seleccionar una comida del menú e imprimir su información (precio, descripción, etc.).
    - Si la contraseña no es válida:
        - Solicitar nuevamente la contraseña. 

Requisitos:

-   No utilizar generadores de código por inteligencia artificial.
-   Utilizar la clase `Scanner` para la entrada de datos.
-   Implementar un bucle para recorrer el arreglo y mostrar el menú.
-   Usar condicionales para la validación de la contraseña y la selección de la comida.
-   Documentar el código con comentarios explicativos.
